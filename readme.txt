Adds a custom badge (Best Seller, Limited Edition, New Arrival) to selected WooCommerce products. Also includes an admin filter and shortcode.

== Description ==
This plugin allows store owners to:
- Add badges to products from the product edit page
- Display the badge on the single product page
- Filter products in admin by badge
- Use the shortcode [custom_badge_products badge="best_seller"] to display filtered product grids

== Installation ==
1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the WordPress admin
3. Edit any WooCommerce product and select a badge

== Shortcode ==
[custom_badge_products badge="new_arrival"]
Displays a grid of products with the selected badge.
